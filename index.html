Connect your app

Initialize OryxaAIHost using your ViewModel and application context. This object bridges your annotated logic with the AI interface. 

// Inside your Activity or Fragment
val host = OryxaAIHost(
    viewModel = viewModel,
    context = requireContext()
)

Detect supported features

The generated metadata from @ExposeToAI allows you to query accessible functions. Useful for debugging or dynamic chat suggestions. 


val features = host.getAvailableFunctions()
features.forEach {
    println("AI Callable: ${it.name}")
}

Add the dependency

Add Oryxa to your build.gradle.kts file:


dependencies {
    implementation("ai.oryxa:oryxa-core:1.0.0")
    ksp("ai.oryxa:oryxa-processor:1.0.0")
}

Enable the KSP plugin:


plugins {
    id("com.google.devtools.ksp") version "1.9.0-1.0.13"
}
